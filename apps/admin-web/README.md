# EventList Admin Web

Admin portal for the EventList workspace.

## Purpose

This app provides administrative workflows for the EventList platform and lives inside the monorepo at `apps/admin-web`.

## Development

From `apps/admin-web`:

```bash
npm install
npm start
```

Or from repository root:

```bash
npm --prefix apps/admin-web install
npm --prefix apps/admin-web start
```

## Build

From `apps/admin-web`:

```bash
npm run build
```

## Notes

- This app is part of the main EventListFull repository.
- Use the root repository remote for collaboration and pushes.
